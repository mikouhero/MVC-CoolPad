# MySQL 常见优化方式   -  优化的范围有哪些
### 存 储、主机和操作系统方面:

    主机架构稳定性 I/O 规划及配置 Swap 交换分区 OS 内核参数和网络问题

### 应用程序方面:

    应用程序稳定性 SQL 语句性能 串行访问资源 性能欠佳会话管理 这个应用适不适合用 MySQL

### 数据库优化方面

    内存 数据库结构 (物理 & 逻辑) 实例配置
    
    说明：不管是在，设计系统，定位问题还是优化，都可以按照这个顺序执行。

### 优化维度
数据库优化维度有四个:
    硬件、系统配置、数据库表结构、SQL 及索引
